发表于2024-08-09|更新于2024-08-09|hexo
|阅读量:

我的个人博客搭建成功啦！
这篇文章将教会大家如何从零开始，快速搭建属于你的个人独立博客。
废话不多说，让我们马上开始！

准备工具

  1. 下载nodejs
  2. 下载git
    • 安装完后请创建你的用户名和邮箱
    • git 的相关文章可参考我的Git学习笔记这篇文章，
  3. 创建github账号
    • 点击右上角Sign up注册账号
    • 用户名会关系到你的默认域名，请慎重！！

为了提升后续资源下载速度，建议配置npm镜像

创建博客文件根目录

  1. 在你喜欢的位置创建一个文件夹，列如:MY_Blog
    以管理员运行cmd,进入当前目录MY_Blog
  2. 下载hexo

搭建仓库

进入github新建一个仓库

  • 点击Create a new repository进入新建仓库页面
  • 仓库名输入
  • 用户名.github.io 用户名就是你github账号用户名
  • 勾选 Public
  • 勾选 Add a README file
  • 拉到下面点击create创建
